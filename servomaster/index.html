ServoMaster

There is a lot of servos used in radio controlled modeling or robotics applications, and there are vendors manufacturing the hardware controllers allowing to control the servos, in particular:
  • FerretTronics FT639, FT649
  • Scott Edwards Electronics, Inc. (Seetron) SSC
  • Quite a lot of vendors providing the hardware controllers based on PIC16F84 device
However, all of them, being in the hardware manufacturing business, care little about providing a consistent interface for the servos and controllers. This is understandable, but disappointing, because all the servos are just about the same, and the basic functionality of the controllers is just about the same.

ServoMaster is a project dedicated to providing a consistent hardware independent way of controlling the servos with the personal computer.
