ServoMaster: Metadata: servo/precision

servo/precision property contains the number of physical steps available between logical 0.0 and 1.0 positions.

Please note that this property may be dynamic for some controllers, an example is USB Phiget family.

As a rule of thumb, if the servo/range/min and servo/range/max properties are writeable, this property's value may change.
