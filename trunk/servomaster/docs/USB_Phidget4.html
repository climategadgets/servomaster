Note
The driver is not finalized yet (though the generic framework requiring minimal changes was put in place), however, the feature set is more or less complete:
  • Improved resolution
  • Supports 8 servos
  • Supports transition control at hardware level - can specify velocity and acceleration
  • Supports servo position feedback - when transition control is active, the actual position is different from requested position
  • Dual (USB/Serial) interface
  • USB bus powered and self powered mode supported
Expected release date is end of September 2002.
