ServoMaster: Quick Start: Easy Way

Prerequisites

  • Java Virtual Machine, version 1.2+
  • Java Communications API properly installed. I would recommend to go the JRE way.

Getting it

Download the servomaster.jar from here (get the latest release).

Running it

You don't really have to have any hardware to run the package. Right now, there is FT639 that is not returning any status, so the driver for it can be run without the hardware.

Run the following command (one line): 

java -jar servomaster.jar
     org.freehold.servomaster.device.impl.ft.FT639ServoController
     <port>
where <port> is COMX on Windows boxes, /dev/ttySX on Linux, /dev/term/X on Solaris and so on. X is the actual port number. Make sure that port is not occupied by your modem, Palm cradle or anything.
