ServoMaster: Metadata: controller/protocol/USB

controller/protocol/USB feature contains true if the hardware supports USB protocol.

Keep in mind that there are controllers that support more than one protocol, like PhidgetAdvancedServo, which will have both controller/protocol/USB and controller/protocol/serial features set to true.

This feature has no relation to the way the hardware is currently connected.
