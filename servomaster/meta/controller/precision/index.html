ServoMaster: Metadata: controller/precision

controller/precision property contains the maximum precision this controller is capable of, in number of steps.

Note that this value may not represent the actual precision exactly - for example, the precision for the particular servos in PhidgetServo will depend on the current min_pulse and max_pulse settings. In order to retrieve the exact precision, use the servo/precision property.
