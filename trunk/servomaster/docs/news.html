Project Status
The Java implementation is rock solid - been working as a part of DIY Zoning for about a year. Drivers are available for
  • FerretTronics FT639
  • Phidget QuadServo
  • Phidget AdvancedServo - any day now

The C++ implementation is breathing and is marginally usable, but by no means complete or stable.

No architecture changes are expected at this time - the project is functionally complete.

Metadata handling is being simplified, however, the existing implementation will be deprecated, not removed.
