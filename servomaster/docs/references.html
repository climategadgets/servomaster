Reference Library
Open Source projects live or fail based on their human resources. Users and contributors alike help the project with ideas and brainpower. A common foundation of knowledge is required to effectively participate in this virtual community. The following is a list of references that we have found helpful for us and may be helpful to you.
Drivers
  • libphidget - a C/C++ implementation of a generic Phidget driver, including servo driver.

Servo Controllers
  • Phidgets (USB servo controller and other good stuff) (driver) - best bang for the buck, with most functionality. Beware: the prices are in canadian dollars!
  • ASC USB Servo Controller - actually, the servo controller is just a part of this device; if you need just the controller, better take Phidget - more functionality, third of the cost
  • Ohmark Digital Servo Controller supports transitions on hardware level
  • FerretTronics FT639 (5 servos) (driver), FT649 (cascading up to 5 FT639s)
  • Scott Edwards Electronics, Inc. Serial Servo Controller (8 servos)
  • Servo Controller with preset PIC 16F84A based (12 servos)
  • Parallel Port Servo Controller
  • Hobby servo controller with torque feedback
  • Pololu 16-servo controller
  • Servo Controller Here's some non-standard thinking
  • 16 channel serial servo controller for robotic applications
  • Micro-Controlled Servo Controller
  • RS-232 Controlled 8/16 Servo Motor Controller
  • Advanced Servo Controller 16 servos, quite advanced but quite expensive
  • SSC-12
  • Servo Controller
  • Pontech SV-203 - 8 servos, SV-203C - same with infrared I/O
  • MIT Mini Board and variation thereof

Dependencies
The following components are necessary to make this work:
  • Ant - Java build tool
  • Java Communications API - required for FT639 (and all serial) driver Java implementation
  • RxTx - to make JavaComm work on Linux
  • jUSB - required for Phidgets driver Java implementation
  • libusb - required for Phidgets C++ implementation
