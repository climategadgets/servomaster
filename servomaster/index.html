Welcome
ServoMaster is dedicated to providing a consistent hardware independent way of controlling the servos with the personal computer.
The Mission
There is a lot of servos used in radio controlled modeling or robotics applications, and there are vendors manufacturing the hardware controllers allowing to control the servos, to name a few:
  • PhidgetServo
  • FerretTronics FT639, FT649
  • Scott Edwards Electronics, Inc. (Seetron) SSC
  • Quite a lot of vendors providing the hardware controllers based on PIC16F84 device
However, almost all of them (with notable exceptions), being in the hardware manufacturing business, care little about providing a consistent interface for the servos and controllers. This is understandable, but disappointing, because all the servos are just about the same, and the basic functionality of the controllers is just about the same.

This can and is being fixed, right here.

Brief History
ServoMaster was created as a complimentary project necessary to satisfy the needs of another project, DIY Zoning. Being lazy (hate to rewrite the same code ten times), I decided to just make it generic from the very beginning. It turned out to be fun. Then, Chester showed up, and it started to be really fun ;)
